Comment est-on identifié ?

Pour s'identifier au système de réservation en ligne, il faut utiliser les mêmes login et mot de passe que pour se connecter au site internet de l'ICA. Si vous n'avez pas de compte sur le site internet de l'ICA vous pouvez contacter l'administrateur du système de réservation.

Pourquoi ne puis-je pas supprimer ou modifier une réservation ?

Pour pouvoir supprimer ou modifier une réservation, vous devez être authentifié comme étant la même personne qui a créé la réservation ou bien un des responsables de l'équipement que vous souhaitez réserver. Contactez soit un des responsables ou la personne qui a créé la réservation pour la supprimer ou la modifier.

Comment créer une réservation qui se répète ?

Après avoir cliqué sur l'heure désirée, l'écran de réservation s'affiche. Choisissez le Type de Périodicité approprié. La salle sera réservée à la même heure, jusqu'à la Date de Fin de Périodicité, et seulement pour les jours spécifiés par le Type de Périodicité.

  • Le type Jour crée une réservation chaque jour.
  • Le type Semaine répète la réservation pour les jours de la semaine que vous avez coché sur la ligne Jour. Par exemple, pour une périodicité hebdomadaire programmée pour chaque lundi, jeudi et vendredi, cochez ces jours sur la ligne Jour. Si vous ne cochez pas de jours, la périodicité aura lieu chaque semaine le même jour que celui initialement choisi.
  • Le type Mois crée une périodicité le même jour chaque mois, par exemple le 15 de chaque mois.
  • Le type Année crée une périodicité le même jour du même mois, par exemple chaque 15 mars.
  • Le type Mois, même jour semaine crée une périodicité un jour par mois, le même jour de la semaine et avec la même position ordinale de la semaine dans le mois. Utilisez ce type pour créer des périodicité du type le premier lundi, le second jeudi ou le quatrième vendredi de chaque mois par exemple. N'utilisez pas ce type pour des réservations après le 28ème jour du mois.
  • Enfin le type tous les n Semaines est le même type que le type semaine mais permet de spécifier un intervalle de semaines, par exemple tous les mardi et jeudi toutes les 3 semaines.

Comment supprimer une seule réservation faisant partie d'une périodicité ?

Choisissez la salle/jour/heure que vous voulez supprimer et choisissez Effacer une réservation. Attention si vous modifiez ensuite la périodicité les réservations supprimées individuellement reviendront.

Comment réserver simultanément des équipements sur différentes cellules ?

Ce n'est pas possible. Pour le moment, vous devez créer les réservations séparément. Assurez vous que les équipements sont disponibles aux même périodes avant de faire la réservation.

Je ne peux pas créer la réservation, j'ai le message trop de réservations!

Il est impossible de créer plus de 365 réservations pour un équipement. Cette limitation est nécessaire pour des raisons de sécurité. Il est possible en cas de besoin d'augmenter cette limite.

Que se passe t'il si plusieurs personnes enregistrent la même réunion ?

Réponse rapide : la première personne qui clique le bouton Enregistrer gagne. En fait, techniquement l'application utilise une base de donnée relationnelle multi-threaded, Multi-Utilisateurs qui peut gérer plusieurs milliers d'utilisateurs simultanés.

Quelle différence entre Maintenance , Formation et Essais ?

Par défaut, un équipement peut être réservé de 3 manières différentes. Maintenance correspond à une période où l'équipement est inaccessible pour cause de maintenance. Formation correspond à un créneau où une formation est prévue sur l'équipement. Renseignez-vous auprès d'un des responsables de l'équipement si vous êtes intéressés par cette formation. Essais correspond à un créneau réservé par un utilisateur pour faire des essais.

Pourquoi dois-je préciser mon établissement d'appartenance quand je réserve un moyen ?

Les frais de maintenance de certains équipements sont partagés. Ainsi, renseigner l'établissement de l'utilisateur permettra de connaître le taux d'occupation du moyen en fonction de chaque établissement pour une répartition des frais plus équitable.

Comment cela fonctionne t'il et qui l'a développé ?

L'application Meeting Room Booking System est un logiciel libre de droits dont les sources sont disponibles, qui est distribué selon la Gnu Public License(GPL). Cela signifie que que vous pouvez modifier, distribuer et utiliser ce logiciel gratuitement.

Cette application est écrite principalement dans le langage PHP, qui est un langage de programmation libre qui peut être mixé dans les pages webs de façon similaire au concept des active server pages de Microsoft. PHP est particulièrement bon dans l'accès aux bases de données.

La base de donnée utilisée par l'application est soit MySQL, soit PostgreSQL. MySQL est un serveur de base base de données SQL (Structured Query Language) robuste, Multi-Utilisateurs, multi-threaded et très rapide qui est aussi sous licence GPL. PostgreSQL est un serveur de base de données relationnelle SQL libre à base d'objets, doté de nombreuses caractéristiques et multi-utilisateurs.

L'application peut fonctionner sur de nombreuses plate-formes, y compris sur des architectures PC utilisant le système d'exploitation Linux.  Linux est un système d'exploitation libre et gratuit, similaire à unix.

Le serveur web utilisé est encore un autre logiciel libre et gratuit. Le serveur web Apache est le serveur web le plus populaire au monde.

En conclusion : chaque composant de ce système, du système d'exploitation à l'application, est complètement gratuit et libre, y compris le code source.
